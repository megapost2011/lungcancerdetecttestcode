# 肺癌量子診断システム - Tkinter版
# Lung Cancer Quantum Diagnosis - Tkinter GUI

Python標準ライブラリTkinterを使用したGUIバージョン

## 🎯 特徴

- **標準ライブラリ**: TkinterはPython標準ライブラリなので追加インストール不要
- **軽量**: PyQt5より軽量で起動が速い
- **クロスプラットフォーム**: Windows、macOS、Linux対応
- **シンプルUI**: 使いやすいインターフェース

## 📦 インストール

### 依存パッケージ

```bash
pip install -r tkinter_requirements.txt
```

Tkinter自体はPython標準ライブラリに含まれています。

### Tkinterの確認

Tkinterが使用可能か確認：

```bash
python -c "import tkinter; print('Tkinter OK')"
```

エラーが出る場合：

**Ubuntu/Debian:**
```bash
sudo apt-get install python3-tk
```

**macOS:**
Pythonに標準で含まれています。

**Windows:**
Pythonに標準で含まれています。

## 🚀 使用方法

### 1. サンプル画像の生成

```bash
python generate_lung_samples.py
```

`lung_ct_samples/`フォルダに11枚のCT画像が生成されます。

### 2. アプリケーションの起動

```bash
python lung_cancer_tkinter.py
```

### 3. 診断の実行

1. **画像を開く**
   - 「📁 画像を開く」ボタンをクリック
   - CT画像を選択（PNG, JPEG, DICOM, NPYファイル対応）

2. **解析モード選択**
   - `clustering`: 結節を3クラスに自動分類
   - `anomaly`: 悪性度の高い結節を検出

3. **解析実行**
   - 「🚀 解析実行」ボタンをクリック
   - 進捗バーで進行状況を確認

4. **結果確認**
   - 入力画像：元のCT画像
   - 処理結果：結節を枠で表示
   - 検出統計：結節数とクラスタ分布
   - クラスタ分布：2次元特徴空間での可視化
   - 検出結節リスト：各結節の詳細情報

5. **結果保存**（オプション）
   - 「💾 結果保存」ボタンでJSON形式で保存

## 🐛 トラブルシューティング

### Tkinterが使えない

**エラー**: `ModuleNotFoundError: No module named 'tkinter'`

**解決方法**:
```bash
# Ubuntu/Debian
sudo apt-get install python3-tk

# Fedora/CentOS
sudo dnf install python3-tkinter
```

### Matplotlibの表示エラー

**エラー**: `TclError: couldn't connect to display`

**解決方法**:
```bash
# X11転送を有効にする（SSH接続時）
export DISPLAY=:0
```

### 処理が遅い

- 画像サイズを小さくする（512×512推奨）
- 量子ビット数を減らす（設定タブ）
- クラスタ数を減らす（設定タブ）
