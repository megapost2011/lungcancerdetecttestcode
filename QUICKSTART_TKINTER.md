# Quick Start Guide - Tkinter Version

## ⚡ 3ステップで始める

### ステップ1: 依存パッケージのインストール

```bash
pip install -r tkinter_requirements.txt
```

**注意**: TkinterはPython標準ライブラリなので、通常は追加インストール不要です。

### ステップ2: サンプル画像の生成

```bash
python generate_lung_samples.py
```

以下の画像が `lung_ct_samples/` に生成されます：
- 正常肺: 3枚
- 良性結節: 3枚
- 悪性結節: 3枚
- 多発結節: 2枚

**合計11枚** のテスト用CT画像

### ステップ3: アプリケーションの起動

```bash
python lung_cancer_tkinter.py
```

## 📝 初めての診断

### 1. 画像を開く
- 「📁 画像を開く」ボタンをクリック
- `lung_ct_samples/malignant_nodule_1.png` を選択

### 2. 解析モード選択
- ドロップダウンから選択
  - **clustering**: おすすめ（結節を3クラスに分類）
  - **anomaly**: 悪性度の高い結節を検出

### 3. 解析実行
- 「🚀 解析実行」ボタンをクリック
- 進捗バーで進行状況を確認（約7秒）

### 4. 結果確認

#### 画像パネル
- **左上（入力画像）**: 元のCT画像
- **左下（処理結果）**: 検出された結節を枠で表示
  - 赤枠: クラスタ0（正常組織）
  - 黄枠: クラスタ1（良性結節）
  - 青枠: クラスタ2（悪性結節疑い）

#### 統計パネル
- **検出統計**: 結節数とクラスタ分布
- **クラスタ分布図**: 2次元特徴空間での可視化
- **検出結節リスト**: 各結節の詳細情報

### 5. 結果保存（オプション）
- 「💾 結果保存」ボタンをクリック
- JSON形式で保存

## 🎮 使い方のコツ

### 良い結果を得るには

1. **高品質な画像を使用**
   - コントラストが明瞭
   - ノイズが少ない
   - 512×512程度の解像度

2. **適切なモード選択**
   - 初めて: `clustering`（自動分類）
   - 詳細分析: `anomaly`（異常検出）

3. **複数画像で比較**
   - 正常肺、良性、悪性を比較
   - クラスタリング傾向を確認

### パフォーマンス改善

処理が遅い場合：

1. **設定タブで調整**
   - 量子ビット数: 6 → 4
   - クラスタ数: 3 → 2

2. **画像サイズを縮小**
   ```python
   # OpenCVでリサイズ
   import cv2
   img = cv2.imread('large.png', cv2.IMREAD_GRAYSCALE)
   small = cv2.resize(img, (512, 512))
   cv2.imwrite('small.png', small)
   ```

## 🐛 よくある問題と解決法

### 問題1: Tkinterが起動しない

**エラー**:
```
ModuleNotFoundError: No module named 'tkinter'
```

**解決法**:
```bash
# Ubuntu/Debian
sudo apt-get install python3-tk

# Fedora
sudo dnf install python3-tkinter
```

### 問題2: 画像が表示されない

**原因**: Matplotlibのバックエンド問題

**解決法**:
```python
# プログラムの先頭に追加
import matplotlib
matplotlib.use('TkAgg')
```

### 問題3: 結節が検出されない

**原因**: 画像の品質やコントラスト

**解決法**:
1. サンプル画像で動作確認
2. 画像の前処理を確認
3. 設定で感度を調整

### 問題4: メモリ不足

**エラー**:
```
MemoryError
```

**解決法**:
1. 画像サイズを縮小
2. 量子ビット数を減らす（設定タブ）
3. 他のアプリケーションを閉じる
